# 📊 Sales Performance Dashboard

## 📌 Project Description

This project is an interactive Excel Dashboard developed using Microsoft Excel for data analysis and visualization. The dashboard transforms raw data into meaningful insights using Pivot Tables, Charts, Slicers, Conditional Formatting, and Excel formulas.

The main objective of this project is to analyze data efficiently and present it in a visually interactive format that helps users understand trends, performance, and key metrics easily.
